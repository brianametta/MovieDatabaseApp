Created by Briana Wright

Personal website to showcase my experience and skills.

Technology used:
    AngularJS
	Mongoose
	Express
	MongoDB
    Bootstrap with Bootswatch "Yeti" theme https://bootswatch.com/yeti/

Setup
    You will need Node.js/npm, MongoDB, Mongoose, and Express
	Run commands in project folder:
	npm install express --save
	npm install mongoose
	node app to start
	
Start of database thanks to tutorial by Traversy Media: 
https://www.youtube.com/watch?v=eB9Fq9I5ocs "RESTful API From Scratch Using Node, Express, and MongoDB"
